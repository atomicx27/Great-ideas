# AGI Curriculum Designer Swarm

The AGI Curriculum Designer Swarm represents the leap into **Artificial General Intelligence (AGI)** within the Education domain. It uses multi-agent orchestration to tackle complex, ambiguous tasks like designing a full curriculum.

Instead of a single prompt, a "Master Orchestrator" receives the high-level educational goal. It then spawns specialized sub-agents (e.g., Mathematics Expert, Science Expert, Literature Expert) to process the requirement simultaneously from different perspectives. Finally, the Master synthesizes their parallel outputs into a cohesive final curriculum.

## Key Features

*   **Multi-Agent Orchestration:** Simulates an AGI brain coordinating multiple specialized models.
*   **Parallel Processing:** Sub-agents work simultaneously, visualized in the UI.
*   **BYOK Architecture:** Supports configurable API connections for major cloud providers or local execution via Ollama.
