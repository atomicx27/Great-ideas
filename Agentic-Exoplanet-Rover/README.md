# Agentic Exoplanet Rover

## Idea
An autonomous rover system deployed on an exoplanet that uses LLM-powered decision making to navigate hazardous terrain, select the most scientifically valuable sampling sites, and adapt its path in real-time.

## How it makes work easier
Rather than waiting for delayed commands from Earth or an orbiting station, the rover can independently analyze complex sensor data and make intelligent decisions to maximize the scientific yield of a mission.
