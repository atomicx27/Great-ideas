# Agentic Submarine Explorer

An autonomous Agentic AI system that navigates deep sea trenches. Given a destination, the agent uses a simulated ReAct loop and tool access (`scanEnvironment`, `adjustDepth`, `propel`) to autonomously navigate, avoid underwater obstacles, and reach the target without human intervention.

## Features
- Simulated ReAct agent loop (Think -> Act -> Observe).
- Tool use simulation for navigation and sensing.
- Autonomous decision making based on environmental observations.
- BYOK (Bring Your Own Key) architecture for LLM integration.
