# Automated Asteroid Scanner

A deterministic automation tool for analyzing spectral data from near-Earth objects. This project relies on predefined thresholds to categorize asteroid composition (e.g., Iron, Water-Ice, Silicate) from raw sensor inputs, demonstrating fundamental, non-AI automation.

## Features
- Simulated spectral data ingestion.
- Rule-based composition categorization.
- Simple reporting dashboard for potential mining viability.
