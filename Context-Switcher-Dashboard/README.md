# Context Switcher Dashboard

## Idea Overview
The Context Switcher Dashboard is a workflow automation tool aimed at mitigating the "context switching tax"—the time and mental energy lost when shifting between completely different tasks or projects. It allows users to create saved "Workspaces." A Workspace bundles relevant links, quick notes, and a specific task list. When an employee switches from "Project A" to "Project B," they simply select the corresponding workspace, and their digital environment instantly reconfigures to provide exactly what they need for that project.

## How it makes work easy and efficient
*   **Zero Setup Time:** Automates the mundane task of opening necessary documents, tools, and links every time you switch tasks.
*   **Reduced Cognitive Load:** You don't have to remember *where* you left off or *what* links you need; the dashboard remembers for you.
*   **Maintained Focus:** By hiding irrelevant information from other projects, it reduces visual clutter and prevents distraction.
*   **Workflow Automation:** Serves as a personal, customizable command center, streamlining how knowledge workers interact with their daily tasks.

## Technologies Used
*   HTML5
*   CSS3
*   JavaScript (Vanilla) utilizing LocalStorage for persistence.
