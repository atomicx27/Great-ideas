# Automated Sonar Mapper

A deterministic automation tool for processing raw sonar data to generate topographic maps of the sea floor. This project does not use AI; it relies on predefined mathematical rules to categorize depths and identify potential hazards, demonstrating the foundational layer of automation.

## Features
- Simulated sonar data ingestion.
- Rule-based depth categorization.
- Hazard detection logic based on rapid depth changes.
- Simple, efficient data processing dashboard.
