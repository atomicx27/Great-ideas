# Automated Meteor Tracker

The Automated Meteor Tracker is a classic deterministic automation tool for Planetary Defense. It uses basic physics and geometry to process sensor inputs (speed, mass, and trajectory angle) to automatically calculate potential Earth impact probabilities. It operates entirely on fixed mathematical thresholds without the use of AI, simulating an early-warning radar system.

### How it makes work easier
By replacing manual trajectory calculations with instant, rule-based alerts, this system ensures that planetary defense operators receive immediate notifications of high-risk space debris or meteors, enabling rapid response times.
