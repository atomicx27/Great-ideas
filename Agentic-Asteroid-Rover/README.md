# Agentic Asteroid Rover

An autonomous AI rover deployed on the surface of an asteroid. Given a mission parameter, the agent uses a simulated ReAct loop and tools (`drillSample`, `analyzeSample`, `moveRover`) to autonomously navigate the rocky terrain, find optimal mining spots, and extract valuable ores.

## Features
- Simulated ReAct agent loop (Think -> Act -> Observe).
- Tool use simulation for movement and drilling.
- Autonomous decision making based on localized sample analysis.
