# Automated-DNA-Sequencer-Monitor

A classic **Automation** tool in the Genomics/Biotech domain. This system uses deterministic rules to continuously monitor incoming data from DNA sequencing runs. It evaluates metrics like read quality scores (Q-scores) and error rates against predefined acceptable thresholds. If metrics drop below quality standards, it automatically alerts lab technicians or pauses the run, preventing wasted reagents without requiring complex AI.
