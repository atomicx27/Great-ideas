# Agentic Interview Conductor

The Agentic Interview Conductor represents a step up into **Agentic AI** in the Human Resources domain. Unlike a static script, this autonomous agent is given a target candidate profile. It autonomously decides which technical topics to cover, dynamically generates questions, evaluates the candidate's simulated responses, and synthesizes the data into a final hiring recommendation.

It supports the standard Bring Your Own Key (BYOK) architecture, allowing integration with OpenAI, Anthropic, or local execution via Ollama.
