# Automated Code Linter

The Automated Code Linter is a classic **Automation** tool in the Software Engineering domain. It is designed to perform static code analysis to enforce coding standards. It does not use AI or LLMs; instead, it relies on predefined, deterministic rules (like checking for missing semicolons, `var` usage, or left-over `console.log` statements) to analyze the code block.

## Key Features

*   **Deterministic Logic:** Fully predictable and immediate results based on strict string parsing rules.
*   **Enforces Standards:** Automatically catches simple anti-patterns before code review.
*   **Fast Execution:** No network calls or complex processing required.
