# Automated Exoplanet Telescope

## Idea
An automated system designed to scan coordinate grids of the sky, calculate light curves for star transits, and identify potential exoplanet candidates based on predefined threshold dips.

## How it makes work easier
It replaces tedious manual inspection of astronomical telemetry data with an efficient, deterministic algorithm that flags interesting star sectors for further human or AI review, drastically reducing the time required to discover new planets.
