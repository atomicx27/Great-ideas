# Agentic Terraforming Rover

**Idea:** An **Agentic AI** application for planetary terraforming. Placed on an alien surface, this autonomous agent uses a ReAct loop and simulated tools (`moveRover`, `analyzeSoil`, `seedSoil`) to search for nutrient-deficient soil and enrich it, adapting its path based on localized soil feedback.

**How it makes work easier:** Instead of a predefined path, the AI dynamically explores and enriches soil, operating independently in hazardous environments where real-time human control is impossible due to light-speed communication delays.
