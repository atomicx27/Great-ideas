# Automated Exam Scorer

The Automated Exam Scorer is a classic **Automation** tool in the Education domain. It is designed to replace manual grading of multiple-choice exams. It does not use AI or LLMs; instead, it relies on deterministic rules and a predefined answer key to automate the grading process.

## Key Features

*   **Error Reduction:** Automated grading reduces human errors in tallying scores.
*   **Speed:** Grades hundreds of submissions instantly.
*   **Deterministic Logic:** Fully predictable results based on strict rules.
