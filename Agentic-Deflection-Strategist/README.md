# Agentic Deflection Strategist

The Agentic Deflection Strategist is a single-agent autonomous system in the Planetary Defense domain. It analyzes high-risk meteor/asteroid data (composition, size, time-to-impact) and autonomously uses Large Language Models to simulate a ReAct loop. It formulates the optimal planetary defense strategy—choosing between kinetic impactors, gravity tractors, or nuclear ablation—based on the specific object constraints.

### How it makes work easier
It acts as an expert defense consultant, synthesizing vast astrophysics constraints dynamically. It saves defense agencies from manually running complex strategic permutations, providing a robust, AI-driven recommendation for saving the planet.
